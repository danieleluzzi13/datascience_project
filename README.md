# datascience_project
Data Science Project
L'analisi del dataset "Access To Electricity" 
Dettagli dei dati: 
● Accesso all'elettricità (% della popolazione) 
● Licenza: Tipo CC BY-4.0 Definizione estesa 
● Accesso all'elettricità (% della popolazione): l'accesso all'elettricità è la percentuale della popolazione che ha accesso all'elettricità. I dati sull'elettrificazione sono raccolti dall'industria, da indagini nazionali e da fonti internazionali. 
● Fonte: Banca mondiale, database Sustainable Energy for All (SE4ALL) del SE4ALL Global Tracking Framework gestito congiuntamente dalla Banca mondiale, dall'Agenzia internazionale per l'energia e dall'Energy Sector Management Assistance Program
Connessioni con l'Obiettivo di Sviluppo Sostenibile 7 "Garantire l'accesso a un'energia economica, affidabile, sostenibile e moderna per tutti". https://sdgs.un.org/goals/goal7
Fonte piattaforma Banca Mondiale: http://databank.worldbank.org/data/source/jobs
